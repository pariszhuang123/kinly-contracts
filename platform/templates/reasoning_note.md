# Reasoning Note

Use this in issues, PRs, and ADRs. No em dashes.

State Assessment
- Current state, assets, constraints.

Target State
- Desired outcome and success criteria.
- How it should feel for users.

Gap Mapping
- Friction points, cognitive and emotional.

Action Decomposition
- Ordered steps with preconditions and effects.
- Rough costs (time/complexity).

Path Planning
- Simple path vs scalable path.
- Dependencies and assumptions.

Adaptive Execution
- Thin slice to ship now.
- Observations to collect.

Reflection Loop
- What worked, failure modes, reusable patterns.

So What for Kinly
- One to three sentences on impact.
