# Kinly Backend API Snapshots

This folder is generated by `kinly-backend` CI and published here automatically.

Contents:
- schema.sql: database schema snapshot
- rls_policies.sql: RLS policy snapshot
- openapi.json: PostgREST OpenAPI snapshot
- types.generated.ts: generated API types
- edge_functions.json: deployed Edge Functions list/metadata
- registry.json: backend contract registry
